# Linktree

Aplicação Front-End desenvolvida com HTML para criação de uma página de links personalizada no estilo Linktree, permitindo centralizar redes sociais, portfólio e outros links importantes em um único lugar.

---

## 📋 Sobre o Projeto

O projeto foi criado com o objetivo de praticar estruturação de páginas web utilizando HTML, desenvolvendo uma interface simples e organizada para compartilhamento de links.

A aplicação funciona como uma página pessoal onde é possível reunir links de redes sociais, contatos, projetos e plataformas digitais.

---

## 🚀 Tecnologias Utilizadas

- HTML5

---

## 🎯 Funcionalidades

- Página de links personalizada
- Organização de links em um único lugar
- Estrutura simples e intuitiva
- Navegação rápida para redes sociais e plataformas
